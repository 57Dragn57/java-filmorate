# Filmorate
Площадка для поиска фильмов и сериалов. Подключайся, находи новых друзей и обсуждайте просмотренное.

# Для пользователя:
1. Просмотр - просматривай ленту или же ищи фильмы по жанрам.
2. Друзья - добавляй друзей и обменивайтесь с ними мнениями.
3. Оценка - ставьте лайк к понравившемуся фильму.
4. Топ - лучшие фильмы/сериалы

# Для развёртывания:
- Java 11+

# В грядущем обновлении планируется:
- Возможность поделиться фильмом с друзьями
- Комментарии к кинокартинам
- Возможность составлять свой ТОП
